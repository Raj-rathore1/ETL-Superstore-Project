# ETL-Superstore-Project

This project demonstrates a complete ETL (Extract, Transform, Load) pipeline using Python and MySQL on real-world sales data.

---

##  Project Overview
The goal of this project is to extract raw sales data, clean and transform it into a structured format, and load it into a MySQL database for further analysis.

---

##  ETL Process

###  Extract
- Loaded raw data from CSV file using Pandas
- Handled encoding issues during data extraction

###  Transform
- Selected relevant columns for analysis
- Cleaned missing values
- Standardized date formats
- Applied data transformations for consistency

###  Load
- Connected to MySQL database
- Created table schema
- Inserted cleaned data into structured database

---

## Tech Stack
- Python (Pandas)
- MySQL
- Jupyter Notebook
